TrueAutoColor

TrueAutoColor is a desktop App made with Electron which uses Ableton Live’s native Api
to create real-time track and clip color changes from track name changes within Ableton Live.
The reason I made this was to solve a pain point for a product which does this exact thing (but worse) while only existing as a plugin, taking away precious cpu from music making.
55+ copies sold and counting!
